Модульні вікна

Цей модуль SwiperComponent призначений для створення, конфігурації та управління слайдерами за допомогою бібліотеки Swiper. Він дозволяє гнучко ініціалізувати слайдер на будь-якому елементі, передаючи параметри за замовчуванням або кастомізовані налаштування.

  • JavaScript
  • HTML

Підключення модуля JS

import Popup from "./components/popups.js";

Створення нового модульного вікна

По кліку на кнопку буде відкриватись модульне вікно яке має id за яким ініціалізовано popup (можна ініціалізувати одразу декілька модульних вікон) 

const popup = new Popup('#popup');

    const openPopupButtons = document.querySelectorAll('.open-popup');
    
    openPopupButtons.forEach(button => {
       button.addEventListener('click', () => {
           popup.openModal();
       });
    });

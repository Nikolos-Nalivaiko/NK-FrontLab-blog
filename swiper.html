Свайпер

Цей модуль SwiperComponent призначений для створення, конфігурації та управління слайдерами за допомогою бібліотеки Swiper. Він дозволяє гнучко ініціалізувати слайдер на будь-якому елементі, передаючи параметри за замовчуванням або кастомізовані налаштування.

  • JavaScript
  • HTML

Підключення модуля JS

import SwiperComponent from './components/swiper.js';

Створення свайперу

Таким чином створюється свайпер із заданням необхідних параметрів, для дочірного свайперу треба писати SwiperThumbs.swiperInstance, де SwiperThumbs - назва створеного дочірного свайпера, а .swiperInstance - для коректної роботи та взаємодії між ними 

const SwiperMain = new SwiperComponent('.swiperMain', {
    slidesPerView: 1,
    spaceBetween: 30,
    speed: 1200,
    autoplay: {
        delay: 5000,
    },
    navigation: {
        prevEl: '.swiper-button-prev',
        nextEl: '.swiper-button-next',
    },

    thumbs: {
        swiper: SwiperThumbs.swiperInstance,
    },
});

Зміна свайперу по кліку

При кліку на якийсь елемент (або якійсь іншій події) можна динамічно змінювати свайпер, а саме змінювати налаштування 

document.querySelector('#update-button').addEventListener('click', () => {
    SwiperMain.updateOptions({
        slidesPerView: 2,
        spaceBetween: 10,
        // інші налаштування
    });
});

Видалення свайперу по кліку

При кліку на якийсь елемент (або якійсь іншій події) можна динамічно "зруйнувати" свайпер 

document.querySelector('#destroy-button').addEventListener('click', () => {
    SwiperMain.destroy();
    SwiperThumbs.destroy();
});

Прелоадер

Легкий та зручний модуль для відображення анімації завантаження під час завантаження веб-сторінки або важливого контенту. Модуль допомагає покращити користувацький досвід, показуючи індикатор завантаження до моменту, коли вся сторінка або її частини будуть повністю готові для взаємодії

  • JavaScript
  • HTML

Підключення модуля JS

import Preloader from "./components/preloader.js";

Ініціалізація модуля

const preloaderElement = document.getElementById('preloader');
const contentElement = document.getElementById('content');

const preloader = new Preloader(preloaderElement, contentElement);

window.onload = function() {
    preloader.hide();
};

Підключення у HTML

В HTML після body потрібно підключити прелоадер через @@include, і додати id=«content» до класу обгортки або до контенту, який не потрібно відображати. Всі стилі прелоадера прописані в HTML шаблоні, щоб уникнути явища FOUC
